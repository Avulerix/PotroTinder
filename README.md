# "PotroTinder"
Este es un proyecto personal creado para la busqueda de pareja dentro de la comunidad de la UAEMex.

Considerar que nada de esto esta anclado a la universidad
Solo fue algo realizado como joda con un amigo.

## Caracteristicas
 - Realizado como un tinder para la comunidad UAEMex.
 - Realizado en PHP puro, sin hacer uso de frameworks o codigo pre-ensamblado.
 - Realizado con las consideraciones de seguridad necesarias.
 - Realizado para tener una nula visibilidad de datos sensibles del usuario.

## Funciones Del Sistema
- Entrada de datos del usuario con formulario POST.
- Inicio de sesion por contraseña.
- Definicion de roles para seguridad de los datos.
- Entrada de imagenes de perfil, descripcion, identificacion.

## Brechas De Seguridad
- La validacion de pertenencia al sistema universitario es realizada po el administrador y de manera manual.
- El sistema es suceptible a spam de solicitudes POST (En el registro o en el inicio de sesion)
